## Commands

### Help
```
* Displays the help menu

* log!help [command-name]
```
### Config
```
* Displays the server configurations.

* log!config
```
### Ping
```
* Displays LogsBot's ping.

* log!ping
```
### Setup
```
* This command is used to set up the logs.

* log!setup option event
```
### Stats
```
* Displays a graph with server stats.

* log!stats option
```
